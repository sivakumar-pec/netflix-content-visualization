# Netflix Content Visualization

Internship project analyzing and visualizing Netflix content data using Microsoft Excel — covering data cleaning, summary statistics, and visual dashboards.

## Objective
To analyze a Netflix content catalog and visualize patterns across content type, genre, country of origin, release year, and content rating.

## Tools Used
- Microsoft Excel
  - Formulas: `COUNTIF`, `COUNTA`, `AVERAGE`
  - Excel Tables
  - Charts: Pie, Bar, Line

## Dataset
A simulated dataset of 600 Netflix titles, modeled on the structure of the popular Kaggle "Netflix Movies and TV Shows" dataset. Fields include:
- show_id, type (Movie / TV Show), title
- country, date_added, release_year
- rating, duration, listed_in (genre)

## Project Structure
- **Raw Data** — cleaned title-level dataset (600 records)
- **Summary** — dashboard sheet containing:
  - Key metrics: Total Titles, Movies, TV Shows, Average Release Year
  - Content Type Breakdown (Movie vs TV Show)
  - Titles by Genre
  - Titles by Country
  - Content Trend by Release Year
  - Titles by Content Rating
  - Visual charts (pie, bar, line)

## Key Insights
- Movies make up a larger share of the catalog compared to TV Shows.
- Content spans a wide range of genres, with a few genres consistently having higher title counts.
- The United States and India are among the top contributing countries by title count.
- Release year trends show content spread across the last two decades, with more recent years contributing more titles.
